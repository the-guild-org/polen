Advanced Queries

Learn how to perform more complex queries with the Pokemon API.

Filtering Pokemon

You can filter Pokemon by various attributes:

query GetFireTypePokemon {
  pokemons(filter: { type: "fire" }) {
    id
    name
    types
  }
}
