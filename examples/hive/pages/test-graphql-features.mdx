## Query with Errors

This query contains types and fields that don't exist in the schema to test error tooltips:

<GraphQLDocumentWithSchema>
{`query TestErrors {
  # These don't exist - should show error tooltips
  nonExistentField
  fakeType {
    anotherFakeField
  }
  
  # This exists
  organization(reference: { slug: "test" }) {
    name
    # This field doesn't exist on Organization
    nonExistentOrgField
  }
}`}
</GraphQLDocumentWithSchema>

All the type names above (like `Organization`, `Project`, `Target`, etc.) should be clickable hyperlinks that navigate to their reference documentation!
