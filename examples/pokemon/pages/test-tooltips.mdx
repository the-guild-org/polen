# Testing GraphQL Document Features

This page tests hover tooltips and copy functionality for GraphQL documents.

## Basic Query with Copy Button

Hover over the identifiers below to see tooltips. Click the copy button in the top-right corner.

```graphql
query GetPokemon {
  pokemon(id: "1") {
    id
    name
    types
  }
}
```

## Query with Variables

Test copying queries with variables:

```graphql
query GetPokemonByName($name: String!) {
  pokemon(name: $name) {
    id
    name
    height
    weight
  }
}
```

## Fragment Example

Complex query with fragments:

```graphql
fragment PokemonDetails on Pokemon {
  id
  name
  types
  height
  weight
}

query GetMultiplePokemon {
  pokemon1: pokemon(id: "1") {
    ...PokemonDetails
  }
  pokemon2: pokemon(id: "4") {
    ...PokemonDetails
  }
}
```

## Mutation Example

```graphql
mutation UpdatePokemon($id: ID!, $name: String!) {
  updatePokemon(id: $id, name: $name) {
    id
    name
    updatedAt
  }
}
```

## Subscription Example

```graphql
subscription OnPokemonUpdate($id: ID!) {
  pokemonUpdated(id: $id) {
    id
    name
    types
    stats {
      hp
      attack
      defense
    }
  }
}
```
