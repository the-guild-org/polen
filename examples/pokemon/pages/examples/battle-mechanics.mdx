# Battle Mechanics

Master the intricacies of Pokemon battles, from moves to type effectiveness!

## Move Details

```graphql interactive
query MoveInformation {
  move(id: "thunderbolt") {
    id
    name
    type
    category  # Physical, Special, or Status
    power
    accuracy
    pp
    priority
    effect
  }
}
```

## Type Effectiveness Chart

Understanding type matchups is crucial. Let's look at Fire-type Pokemon:

```graphql interactive
query FireTypePokemon {
  pokemonByType(type: FIRE) {
    id
    name
    types
    stats {
      attack
      specialAttack
      defense
      specialDefense
      speed
    }
    moves {
      name
      type
      power
      category
    }
  }
}
```

## Battle Information

View active battles and their status:

```graphql interactive
query ActiveBattles {
  activeBattles {
    id
    format
    weather
    terrain
    turns
    trainer1 {
      name
    }
    trainer2 {
      name
    }
    wildPokemon {
      name
      currentHp
      status
    }
  }
}
```

## Pokemon Abilities

Explore special abilities that affect battles:

```graphql interactive
query AbilityInfo {
  levitate: ability(id: "levitate") {
    id
    name
    description
    battleEffect
  }
  
  levitators: pokemonByAbility(abilityId: "levitate") {
    name
    types
    abilities {
      name
    }
  }
}
```
