# Trainer Journeys

Follow the adventures of Pokemon trainers, their teams, and achievements!

> **Note:** Trainers were introduced in v2.0.0 of the Pokemon API

## Trainer Profile (v2.0.0+)

Basic trainer information:

```graphql interactive
query TrainerProfile {
  trainer(id: "ash-ketchum") {
    id
    name
    hometown
    badges
    startDate
  }
}
```

## Trainer's Pokemon Team (v2.0.0+)

See a trainer's current team composition:

```graphql interactive
query TrainerTeam {
  trainer(id: "misty") {
    name
    pokemon {
      name
      types
      level
      isShiny
      moves {
        name
        type
        power
        accuracy
      }
    }
  }
}
```

## Advanced Trainer Data (v3.0.0+)

v3.0.0 adds trainer classes, inventory, and battle statistics:

```graphql interactive
query AdvancedTrainerProfile {
  trainer(id: "brock") {
    name
    class
    money
    pokedexCompletion
    
    # Battle record
    battleStats {
      wins
      losses
      draws
      winRate
    }
    
    # Inventory
    items {
      item {
        name
        category
        description
      }
      quantity
    }
  }
}
```

## Catching Pokemon (v2.0.0+)

Use mutations to catch Pokemon:

```graphql interactive
mutation CatchPokemon {
  catchPokemon(
    pokemonId: "025"
    trainerId: "ash-ketchum"
  ) {
    id
    name
    trainer {
      name
    }
  }
}
```

## Trading Pokemon (v3.0.0+)

v3.0.0 introduces Pokemon trading:

```graphql interactive
mutation TradePokemon {
  tradePokemon(
    pokemonId: "025"
    fromTrainerId: "ash-ketchum"
    toTrainerId: "misty"
  ) {
    pokemon {
      name
    }
    fromTrainer {
      name
    }
    toTrainer {
      name
    }
    triggeredEvolution
  }
}
```

## Real-time Updates (v3.0.0+)

Subscribe to trainer events:

```graphql interactive
subscription WatchTrainerProgress {
  pokemonLevelUp(trainerId: "ash-ketchum") {
    name
    level
    evolution {
      evolvesTo {
        name
      }
    }
  }
}
```
