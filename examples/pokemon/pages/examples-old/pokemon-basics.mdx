# Pokemon Basics

Learn how to query individual Pokemon and their core attributes. Hover over any field to see its type information!

## Fetching a Single Pokemon

The most basic query - get a Pokemon by ID:

```graphql interactive
query GetPikachu {
  pokemon(id: "025") {
    id
    name
    type    # In v1.0.0
  }
}
```

## Querying Multiple Types (v2.0.0+)

Starting in v2.0.0, Pokemon can have multiple types:

```graphql interactive  
query GetCharizard {
  pokemon(id: "006") {
    id
    name
    types   # Array of types in v2.0.0+
  }
}
```

## Pokemon Stats

### Basic Stats (v1.0.0)

In v1.0.0, Pokemon have simple combat stats:

```graphql interactive
query BasicStats {
  pokemon(id: "150") {  # Mewtwo
    name
    hp
    attack
    defense
  }
}
```

### Advanced Stats (v2.0.0+)

From v2.0.0 onwards, stats are grouped in a Stats object:

```graphql interactive
query AdvancedStats {
  pokemon(id: "150") {  # Mewtwo
    name
    stats {
      hp
      attack
      defense
      specialAttack
      specialDefense
      speed
    }
  }
}
```

## Pokemon with Abilities (v2.0.0+)

Each Pokemon has unique abilities that affect battles:

```graphql interactive
query PokemonAbilities {
  pokemon(id: "006") {  # Charizard
    name
    abilities {
      name
      description
      isHidden
    }
  }
}
```

## Advanced Pokemon Data (v3.0.0+)

v3.0.0 adds genetic values, training data, and more:

```graphql interactive
query DetailedPokemonInfo {
  pokemon(id: "001") {
    name
    types
    level
    friendship
    nature
    
    # Genetic potential
    ivs {
      hp
      attack
      defense
      specialAttack
      specialDefense
      speed
    }
    
    # Training results
    evs {
      hp
      attack
      defense
      specialAttack
      specialDefense
      speed
    }
  }
}
```

## Listing Pokemon

### Basic List (v1.0.0)

```graphql interactive
query ListPokemons {
  pokemons(type: FIRE) {
    id
    name
    type
  }
}
```

### Paginated List (v3.0.0+)

v3.0.0 introduces proper pagination:

```graphql interactive
query PaginatedPokemons {
  pokemons(limit: 5, offset: 0) {
    edges {
      node {
        id
        name
        types
        level
      }
    }
    pageInfo {
      hasNextPage
      totalCount
    }
  }
}
```
